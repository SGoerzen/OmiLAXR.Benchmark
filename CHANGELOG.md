# Changelog

## v1.0.0

First public release of OmiLAXR.Benchmark.

Highlights:
- Benchmark runner with warmup + measurement workflow.
- CSV logging with time-based bins (p95/max) plus log reports.
- Stress scenario prefab with deterministic motion and optional synthetic events.
- Sample scenes for baseline vs OmiLAXR tracking comparison.
- Python plotting helpers in `Samples~`.
